Configuring KrakenD Documentation

The KrakenD has the following system-related configuration properties:

  • port: The http port. Default 8080
  • stats_port: The stats port. Default 8082
  • ssl_port: The https port. Default 8081
  • ssl_certificate: The path to the ssl certificate for the https service.
  • ssl_private_key: The path to the private key for the https service.
  • global_max_rate: Max rate for the entire service, aggregating over all the exposed endpoints (in request per second)
  • max_rate_per_endpoint: Max rate by for every exposed endpoints (in request per second)
  • log: Log level. Posible values: ERROR, WARNING, INFO, DEBUG. Default: ERROR
  • syslog: Send logs to the syslog. Default value false
  • stdout: Send logs to the stdout. Default value true
  • config_file: Path to the endpoint configuration file (usually a json). Default: /etc/krakend/krakend.json
  • max_idle_conns_per_host: Number of idle connections to keep alive per host. Default: 250
  • read_timeout: http read timeout. Check this picture for a quick understanding: https://blog.cloudflare.com/content/images/2016/06/Timeouts-001.png
  • write_timeout: http write timeout. Check this picture for a quick understanding: https://blog.cloudflare.com/content/images/2016/06/Timeouts-001.png

These configuration values can be defined in the service.yml file and some of them, also as flags to the CLI (check the Using KrakenD section). KrakenD will look for the YAML config file at /etc/krakend/service.yml and update all the service config parameters present in the file.

Sample service.yml file:

# port : 8080
# stats_port : 8082
# ssl_port : 8081

# ssl_certificate : "/etc/krakend/.ssh/server.rsa.crt"
# ssl_private_key : "/etc/krakend/.ssh/server.rsa.key"

# global_max_rate : 500 # req/s
# max_rate_per_endpoint : 10 # req/s

# log : ERROR
# syslog : false
# stdout : true

# config_file : "/etc/krakend/krakend.json"

max_idle_conns_per_host : 250
read_timeout : 3s
write_timeout : 3s

Sample config overwrite:

$ krakend run --config $HOME/krakend.json --stats_port 8090 --ssl_certificate $HOME/server.rsa.crt --ssl_private_key $HOME/server.rsa.key --log WARNING --global_max_rate 150
