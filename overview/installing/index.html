RPM installation

The installation process requires following these steps:

  1. Install the repo package
  2. Install the KrakenD package
  3. Start the KrakenD service

It can be done with this:

$ rpm -Uvh http://repo.krakend.io/rpm/krakend-repo-0.1-0.noarch.rpm
$ yum install -y krakend
$ systemctl start krakend

Since fedora has deprecated yum in favor of dnf, you must replace one for the other (btw, dnf will start your KrakenD service):

$ rpm -Uvh http://repo.krakend.io/rpm/krakend-repo-0.1-0.noarch.rpm
$ dnf install -y krakend
$ systemctl start krakend

The process has been tested on centos 7 & fedora 24

DEB installation

The installation process requires following these steps:

  1. Add the key
  2. Add the repo to the sources.list
  3. Update your package list
  4. Install the KrakenD service

Bottom line:

$ apt-key adv --keyserver keyserver.ubuntu.com --recv AB39BEA1
$ echo "deb http://repo.krakend.io/apt stable main" | tee /etc/apt/sources.list.d/krakend.list
$ apt-get update
$ apt-get install -y krakend

The process has been tested on debian 8 and ubuntu 16.x

Manual installation

  1. Download the binary for your architecture. Check the Download section for the url.
  2. Create your /etc/krakend/service.yml config file
  3. Create your /etc/krakend/krakend.json config file
  4. Start the KrakenD: ./krakend run
